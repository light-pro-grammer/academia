## Що таке система лінійних рівнянь?

**Система лінійних рівнянь** — два або більше рівняння які треба виконати одночасно.
```text
2x + y = 7
x - y = 2
```

Розв'язок системи — пара (x; y) яка задовольняє обидва рівняння.

## Геометричний зміст

Кожне рівняння ax + by = c задає **пряму** на площині.

Розв'язок системи — **точка перетину** цих прямих.

Три випадки:
- Прямі перетинаються в одній точці → **один розв'язок**
- Прямі паралельні → **немає розв'язків**
- Прямі збігаються → **нескінченно багато розв'язків**

## Метод підстановки

1. Виразити одну змінну через іншу з одного рівняння
2. Підставити у друге рівняння
3. Розв'язати рівняння з однією змінною
4. Знайти другу змінну

**Приклад:**
```text
2x + y = 7
x - y = 2
```

З другого рівняння: `x = y + 2`

Підставляємо в перше: `2(y + 2) + y = 7`
`2y + 4 + y = 7`
`3y = 3`
`y = 1`

Тоді `x = 1 + 2 = 3`

Відповідь: **(3; 1)**

Перевірка: `2*3 + 1 = 7` ✓ і `3 - 1 = 2` ✓

## Метод додавання (виключення)

1. Множимо рівняння на числа щоб коефіцієнти однієї змінної стали протилежними
2. Додаємо рівняння — змінна зникає
3. Розв'язуємо рівняння з однією змінною

**Приклад:**
```text
3x + 2y = 12
x - 2y = 4
```

Коефіцієнти при y вже протилежні (2 і -2). Додаємо:
`3x + 2y + x - 2y = 12 + 4`
`4x = 16`
`x = 4`

Підставляємо: `3*4 + 2y = 12 → 2y = 0 → y = 0`

Відповідь: **(4; 0)**

## Метод Крамера (для 2×2)

Для системи:
```text
a₁x + b₁y = c₁
a₂x + b₂y = c₂
```

Визначник системи: D = a₁*b₂ - a₂*b₁

x = (c₁*b₂ - c₂*b₁) / D
y = (a₁*c₂ - a₂*c₁) / D

Якщо D = 0 — система або не має розв'язків або має нескінченно багато.

## Практичні задачі

**Задача 1 — суміш:**
Є два сплави: перший містить 30% міді, другий — 70%. Скільки кг кожного треба змішати щоб отримати 100 кг сплаву з 50% міді?
```text
x + y = 100
0.3x + 0.7y = 50
```

З першого: `x = 100 - y`
Підставляємо: `0.3(100-y) + 0.7y = 50`
`30 - 0.3y + 0.7y = 50`
`0.4y = 20`
`y = 50`, `x = 50`

**Задача 2 — рух:**
Два міста на відстані 300 км. Звідти виїхали два автомобілі назустріч. Перший зі швидкістю v₁, другий — v₂. Зустрілися через 2 години. Перший їхав на 20 км/год швидше.
```text
v₁ + v₂ = 150
v₁ - v₂ = 20
```

Додаємо: `2v₁ = 170 → v₁ = 85 км/год`
`v₂ = 150 - 85 = 65 км/год`

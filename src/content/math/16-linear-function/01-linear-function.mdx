## Означення

**Лінійна функція** — функція виду:

**f(x) = kx + b**

де k і b — дійсні числа, k != 0.

Графік лінійної функції — **пряма лінія** (звідси назва).

## Параметри k і b

**k — кутовий коефіцієнт:**
- Показує нахил прямої
- При `k > 0` — пряма зростає (іде вгору зліва направо)
- При `k < 0` — пряма спадає (іде вниз зліва направо)
- Чим більше |k| — тим крутіший нахил

**b — вільний член (зміщення):**
- Показує де пряма перетинає вісь Y
- f(0) = b — точка перетину з віссю Y завжди (0; b)

## Побудова графіка

Для прямої достатньо **двох точок**.

**Приклад: f(x) = 2x - 3**

Беремо два зручних x:
- x = 0: f(0) = -3 → точка (0; -3)
- x = 2: f(2) = 4 - 3 = 1 → точка (2; 1)

З'єднуємо точки прямою.

## Нуль функції

Нуль лінійної функції — значення x при якому f(x) = 0:

kx + b = 0
x = -b/k

**Приклад: f(x) = 3x - 6**
x = 6/3 = **2**

Графік перетинає вісь X в точці (2; 0).

## Властивості

- **Область визначення:** всі дійсні числа
- **Область значень:** всі дійсні числа
- **Монотонність:** при k `>` 0 зростає на всій числовій прямій, при k `<` 0 спадає
- **Нулі:** рівно один нуль x = -b/k

## Паралельні і перпендикулярні прямі

**Паралельні прямі** мають однакові кутові коефіцієнти:
- y = 2x + 1 і y = 2x - 5 — паралельні (обидві k = 2)

**Перпендикулярні прямі** — добуток їх кутових коефіцієнтів = -1:
- k₁ × k₂ = -1
- y = 2x + 1 і y = -x/2 + 3 — перпендикулярні (2 × (-1/2) = -1)

## Рівняння прямої через дві точки

Якщо відомі дві точки (x₁; y₁) і (x₂; y₂):

k = (y₂ - y₁) / (x₂ - x₁)

**Приклад:** пряма через (1; 3) і (3; 7):
k = (7 - 3) / (3 - 1) = 4/2 = 2
y = 2x + b → 3 = 2*1 + b → b = 1
Відповідь: **y = 2x + 1**

## Застосування

📈 **Економіка:** витрати = фіксована частина + змінна частина
C(x) = 500 + 50x (500 грн фіксовано + 50 грн за одиницю)

🚗 **Рух:** s = v*t (шлях як лінійна функція часу при постійній швидкості)

🌡️ **Перетворення температур:** F = 1.8*C + 32 (Цельсій у Фаренгейт)

## Що таке графік функції?

**Графік функції** f — це множина всіх точок (x, f(x)) на координатній площині.

Кожна точка графіка має координати: x — аргумент, y = f(x) — значення функції.

## Як побудувати графік?

1. Складаємо таблицю значень для кількох x
2. Наносимо точки на координатну площину
3. З'єднуємо точки плавною кривою

**Приклад: f(x) = 2x + 1**

| x | -2 | -1 | 0 | 1 | 2 |
|---|----|----|---|---|---|
| f(x) | -3 | -1 | 1 | 3 | 5 |

Точки: (-2; -3), (-1; -1), (0; 1), (1; 3), (2; 5) — пряма лінія.

## Читання графіка

З графіка можна зчитувати:

**Значення функції:** знайди x на осі X, підніміться вертикально до графіка, прочитай y.

**Нулі функції:** точки де графік перетинає вісь X (y = 0).

**Знак функції:** де графік вище осі X — функція додатна, нижче — від'ємна.

**Зростання/спадання:** графік іде вгору зліва направо — функція зростає, вниз — спадає.

## Основні графіки

**Лінійна функція f(x) = kx + b** — пряма лінія.
- k — кутовий коефіцієнт (нахил)
- b — зміщення по осі Y
- При k `>` 0 — зростає, при k `<` 0 — спадає

**Квадратична функція f(x) = x²** — парабола.
- Вершина в точці (0, 0)
- Симетрична відносно осі Y
- Зростає при x `>` 0, спадає при x `<` 0

**f(x) = 1/x** — гіпербола.
- Не визначена при x = 0
- Два окремих гілки у I і III квадрантах

**f(x) = |x|** — модуль.
- V-подібний графік
- Вершина в (0, 0)

## Властивості функції за графіком

**Область визначення** — проекція графіка на вісь X.

**Область значень** — проекція графіка на вісь Y.

**Парна функція** — графік симетричний відносно осі Y. f(-x) = f(x). Приклад: f(x) = x².

**Непарна функція** — графік симетричний відносно початку координат. f(-x) = -f(x). Приклад: f(x) = x³.

**Монотонна функція** — або тільки зростає або тільки спадає на всій області визначення.

## Зсуви графіків

Якщо знаєш графік f(x) — легко будуєш похідні:

- **f(x) + c** — зсув вгору на c
- **f(x) - c** — зсув вниз на c
- **f(x + c)** — зсув вліво на c
- **f(x - c)** — зсув вправо на c
- **-f(x)** — відображення відносно осі X
- **f(-x)** — відображення відносно осі Y

**Приклад:** f(x) = (x - 2)² + 3 — це парабола x² зсунута на 2 вправо і 3 вгору. Вершина в (2; 3).

## Функція — одна з найважливіших ідей математики

Функція описує **залежність** між двома величинами. Це один з найпотужніших інструментів математики — без функцій немає фізики, програмування, економіки чи машинного навчання.

## Означення

**Функція f : X → Y** — це правило яке кожному елементу множини X ставить у відповідність **рівно один** елемент множини Y.

- **X** — область визначення (domain) — допустимі вхідні значення
- **Y** — область значень (range) — можливі вихідні значення
- **f(x)** — значення функції в точці x (читається "f від x")

## Ключова умова

Кожному x відповідає **рівно один** y.

Якщо одному x відповідає два різних y — це не функція.

Приклад НЕ функції: x² + y² = 25 (окружність) — одному x відповідають два значення y.

## Способи задання функції

**1. Формулою:**
f(x) = 2x + 3
f(x) = x²
f(x) = √x

**2. Таблицею:**

| x | 1 | 2 | 3 | 4 |
|---|---|---|---|---|
| f(x) | 5 | 7 | 9 | 11 |

**3. Графіком:**
Графік функції — множина точок (x, f(x)) на координатній площині.

**4. Словесно:**
"Площа квадрата зі стороною x" → f(x) = x²

## Обчислення значень функції

Якщо f(x) = 3x² − 2x + 1:

f(0) = 3(0)² − 2(0) + 1 = **1**
f(2) = 3(4) − 2(2) + 1 = 12 − 4 + 1 = **9**
f(−1) = 3(1) − 2(−1) + 1 = 3 + 2 + 1 = **6**

## Область визначення

**Область визначення** — всі допустимі значення x.

За замовчуванням — всі дійсні числа. Але є обмеження:

**f(x) = 1/x:** x ≠ 0 (ділення на нуль заборонено)
D(f) = (−∞; 0) ∪ (0; +∞)

**f(x) = √x:** x ≥ 0 (корінь з від'ємного не існує)
D(f) = [0; +∞)

**f(x) = √(x − 3):** x − 3 ≥ 0 → x ≥ 3
D(f) = [3; +∞)

**f(x) = 1/(x² − 4):** x² ≠ 4 → x ≠ ±2
D(f) = (−∞; −2) ∪ (−2; 2) ∪ (2; +∞)

## Функції навколо нас

📱 **Програмування:**
Функція в коді — те саме що функція в математиці. Отримує вхід, повертає вихід.
```python
def square(x):
    return x ** 2
```

🌡️ **Фізика:**
Швидкість як функція часу: v(t) = v₀ + at

💹 **Економіка:**
Попит як функція ціни: D(p) = 100 − 2p

🧬 **Біологія:**
Ріст популяції: N(t) = N₀ × eʳᵗ

## Вертикальний тест

Простий спосіб перевірити чи є графік функцією:

**Вертикальний тест:** якщо будь-яка вертикальна пряма перетинає графік не більше ніж в одній точці — це функція.

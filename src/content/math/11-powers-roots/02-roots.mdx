## Що таке корінь?

**Корінь** — це операція обернена до піднесення до степеня.

Якщо 3² = 9, то √9 = 3.
Якщо 2³ = 8, то ∛8 = 2.

Питання яке ставить корінь: **яке число треба звести у степінь щоб отримати дане?**

## Квадратний корінь

**√a** — квадратний корінь з a — це число яке у квадраті дає a.

√9 = 3, бо 3² = 9
√16 = 4, бо 4² = 16
√2 ≈ 1,414...

**Важливо:** √a визначений лише для a ≥ 0 (у дійсних числах).
√(-4) — не існує у дійсних числах.

## Кубічний корінь

**∛a** — кубічний корінь з a — це число яке у кубі дає a.

∛8 = 2, бо 2³ = 8
∛27 = 3, бо 3³ = 27
∛(-8) = -2, бо (-2)³ = -8

На відміну від квадратного — кубічний корінь існує і для від'ємних чисел!

## Корінь n-го степеня

**ⁿ√a** — корінь n-го степеня з a.

⁴√16 = 2, бо 2⁴ = 16
⁵√32 = 2, бо 2⁵ = 32

## Ірраціональні числа

Квадратний корінь з числа яке не є точним квадратом — **ірраціональне число**: нескінченний дріб без закономірності.

√2 = 1,41421356...
√3 = 1,73205080...
√5 = 2,23606797...

Ірраціональні числа не можна записати як дріб p/q.

## Властивості квадратних коренів

**Множення:** √a × √b = √(ab)
- √4 × √9 = √36 = 6
- Перевірка: 2 × 3 = 6 ✓

**Ділення:** √a ÷ √b = √(a/b)
- √16 ÷ √4 = √4 = 2

**Корінь квадрата:** √(a²) = |a|
- √((-3)²) = √9 = 3 = |-3|

**Квадрат кореня:** (√a)² = a
- (√5)² = 5

## Спрощення коренів

√12 = √(4 × 3) = √4 × √3 = 2√3
√50 = √(25 × 2) = √25 × √2 = 5√2
√72 = √(36 × 2) = √36 × √2 = 6√2

## Застосування у фізиці та математиці

**Теорема Піфагора:**
c = √(a² + b²)
Гіпотенуза прямокутного трикутника з катетами 3 і 4:
c = √(9 + 16) = √25 = **5**

**Швидкість звільнення з гравітаційного поля:**
v = √(2gR) ≈ 11,2 км/с для Землі

**Стандартне відхилення у статистиці:**
σ = √(Σ(xᵢ - μ)²/n)

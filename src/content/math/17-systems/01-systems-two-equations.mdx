## Системи рівнянь — глибше

У розділі про лінійну функцію ми вже познайомились з системами. Тут розглянемо їх детальніше і розширимо на нелінійні системи.

## Методи розв'язання — повторення

**Метод підстановки** — виразити одну змінну через іншу.

**Метод додавання** — підібрати множники щоб одна змінна зникла.

**Метод Крамера** — через визначники (для лінійних систем).

## Системи з нелінійними рівняннями

Системи можуть містити квадратні рівняння, корені тощо.

**Приклад:**
```text
x + y = 5
x * y = 6
```

З першого: `y = 5 - x`
Підставляємо: `x(5 - x) = 6`
`5x - x² = 6`
`x² - 5x + 6 = 0`
`(x-2)(x-3) = 0`
x = 2 або x = 3

При x = 2: y = 3
При x = 3: y = 2

Відповідь: **(2; 3)** і **(3; 2)**

## Система з квадратним рівнянням
```text
y = x²
y = x + 2
```

Підставляємо: `x² = x + 2`
`x² - x - 2 = 0`
`(x-2)(x+1) = 0`
x = 2 або x = -1

При x = 2: y = 4
При x = -1: y = 1

Відповідь: **(2; 4)** і **(-1; 1)**

Геометрично — точки перетину параболи і прямої.

## Система рівнянь з модулями
```text
|x| + y = 4
x - y = 0
```

З другого: `x = y`

Підставляємо: `|x| + x = 4`

Випадок 1: x `>=` 0 → x + x = 4 → x = 2, y = 2
Випадок 2: x `<` 0 → -x + x = 4 → 0 = 4 — суперечність

Відповідь: **(2; 2)**

## Практичні задачі

**Задача — прямокутник:**
Периметр прямокутника 28 см, площа 48 см². Знайди сторони.
```text
2(x + y) = 28  →  x + y = 14
x * y = 48
```

З першого: y = 14 - x
`x(14 - x) = 48`
`x² - 14x + 48 = 0`
D = 196 - 192 = 4
x = (14 ± 2) / 2 → x = 8 або x = 6

Сторони: **6 см і 8 см**

**Задача — числа:**
Сума квадратів двох чисел 25, сума чисел 7. Знайди числа.
```text
x² + y² = 25
x + y = 7
```

z другого: y = 7 - x
`x² + (7-x)² = 25`
`x² + 49 - 14x + x² = 25`
`2x² - 14x + 24 = 0`
`x² - 7x + 12 = 0`
x = 3 або x = 4

Числа: **3 і 4**

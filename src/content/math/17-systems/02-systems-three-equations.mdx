## Системи трьох рівнянь з трьома невідомими

Система трьох рівнянь:
```text
a₁x + b₁y + c₁z = d₁
a₂x + b₂y + c₂z = d₂
a₃x + b₃y + c₃z = d₃
```

Розв'язок — трійка (x; y; z) яка задовольняє всі три рівняння.

## Метод послідовного виключення

1. Виключаємо одну змінну з двох пар рівнянь
2. Отримуємо систему двох рівнянь з двома невідомими
3. Розв'язуємо її
4. Знаходимо третю змінну

**Приклад:**
```text
x + y + z = 6
2x - y + z = 3
x + 2y - z = 5
```

Крок 1 — виключаємо z. Додаємо рівняння 1 і 3:
`(x+y+z) + (x+2y-z) = 6+5`
`2x + 3y = 11` ... (A)

Додаємо рівняння 2 і 3:
`(2x-y+z) + (x+2y-z) = 3+5`
`3x + y = 8` ... (B)

Крок 2 — розв'язуємо систему A і B:
З (B): `y = 8 - 3x`
Підставляємо в (A): `2x + 3(8-3x) = 11`
`2x + 24 - 9x = 11`
`-7x = -13`
`x = 13/7`... 

Спробуємо простіший приклад:
```text
x + y + z = 6
x - y + z = 2
x + y - z = 4
```

Рівняння 1 - Рівняння 2: `2y = 4 → y = 2`
Рівняння 1 - Рівняння 3: `2z = 2 → z = 1`
З рівняння 1: `x = 6 - 2 - 1 = 3`

Відповідь: **(3; 2; 1)**

Перевірка: 3+2+1=6 ✓, 3-2+1=2 ✓, 3+2-1=4 ✓

## Геометричний зміст

Кожне рівняння ax + by + cz = d задає **площину** у тривимірному просторі.

Три площини можуть:
- Перетинатися в одній точці → **один розв'язок**
- Мати спільну пряму → **нескінченно багато**
- Не мати спільних точок → **немає розв'язків**

## Метод Крамера для 3×3

Для системи трьох рівнянь визначник:
```
|a₁ b₁ c₁|
D = |a₂ b₂ c₂|
|a₃ b₃ c₃|
```

D = a₁(b₂c₃ - b₃c₂) - b₁(a₂c₃ - a₃c₂) + c₁(a₂b₃ - a₃b₂)

Якщо D != 0 — система має єдиний розв'язок.

## Практична задача

Три товари коштують разом 100 грн.
Перший і другий разом — 70 грн.
Другий і третій разом — 80 грн.
Знайди ціну кожного.
```text
x + y + z = 100
x + y = 70
y + z = 80
```

З рівнянь 1 і 2: `z = 30`
З рівняння 3: `y = 80 - 30 = 50`
З рівняння 2: `x = 70 - 50 = 20`

Відповідь: x = **20 грн**, y = **50 грн**, z = **30 грн**

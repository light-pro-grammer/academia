## Що таке система нерівностей?

**Система нерівностей** — це кілька нерівностей які повинні виконуватись **одночасно**.

Розв'язок системи — множина значень x які задовольняють **всі** нерівності системи.

Приклад системи:
```text
2x + 1 > 5
x - 3 <= 4
```

## Алгоритм розв'язання

1. Розв'яжи кожну нерівність окремо
2. Знайди **перетин** отриманих множин
3. Запиши відповідь

## Приклад 1
```text
x > 2
x < 7
```

- Перша нерівність: `x > 2`
- Друга нерівність: `x < 7`
- Перетин: `2 < x < 7` або x належить (2; 7)

## Приклад 2
```text
3x - 1 > 5
2x + 3 <= 11
```

- `3x - 1 > 5`, тому `3x > 6`, тому `x > 2`
- `2x + 3 <= 11`, тому `2x <= 8`, тому `x <= 4`
- Перетин: `2 < x <= 4` або x належить (2; 4]

## Приклад 3 — порожній розв'язок
```text
x > 5
x < 3
```

Числа більші за 5 і менші за 3 одночасно — таких немає. **Система не має розв'язків**.

## Приклад 4 — одна нерівність поглинає іншу
```text
x > 1
x > 4
```

Перетин — більш жорстка умова: `x > 4`

## Подвійна нерівність як система

`-1 < 2x + 3 <= 9` означає:
```text
2x + 3 > -1
2x + 3 <= 9
```

- `2x > -4`, тому `x > -2`
- `2x <= 6`, тому `x <= 3`
- Відповідь: `-2 < x <= 3` або x належить (-2; 3]

## Інтервальний запис

- (a; b) — `a < x < b` — відкритий інтервал
- [a; b] — `a <= x <= b` — закритий відрізок
- [a; b) — `a <= x < b` — напіввідкритий
- (a; +inf) — `x > a`
- (-inf; b] — `x <= b`

## Практичні задачі

Реакція при температурі від 20 до 80°C: `20 <= T <= 80`

Книга 75 грн. Скільки книг на суму від 200 до 500 грн?
`200 <= 75n <= 500`, тому `2.67 <= n <= 6.67`. Оскільки n ціле: **від 3 до 6 книг**.

Ліфт витримує від 200 до 600 кг. Середня маса 75 кг:
`200 <= 75n <= 600`, тому `2.67 <= n <= 8`. **Від 3 до 8 людей**.

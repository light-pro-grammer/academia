## Нерівності vs рівняння

Рівняння `2x + 3 = 11` дає точне значення x.
Нерівність `2x + 3 > 11` дає множину значень x.

**Нерівність** — математичний запис що стверджує що один вираз більший, менший або не рівний іншому.

Знаки нерівностей: більше `>`, менше `<`, більше або рівне `>=`, менше або рівне `<=`, не рівне `!=`

## Розв'язок нерівності

**Розв'язок нерівності** — множина всіх значень змінної при яких нерівність виконується.

На відміну від рівняння — зазвичай нескінченно багато розв'язків.

Наприклад: `x > 3` — всі числа більші за 3: 3.1, 4, 7, 100...

## Зображення на числовій прямій

- `x > 3` — відкрита точка на 3, стрілка вправо (3 не входить)
- `x >= 3` — закрита точка на 3, стрілка вправо (3 входить)
- `x < 3` — відкрита точка на 3, стрілка вліво
- `2 <= x < 7` — відрізок від 2 включно до 7 не включно

## Властивості нерівностей

**Властивість 1 — додавання/віднімання:**
Якщо `a > b`, то `a + c > b + c` для будь-якого c.
Приклад: `7 > 4`, тому `7 + 3 > 4 + 3`, тобто `10 > 7`.

**Властивість 2 — множення на додатне число:**
Якщо `a > b` і `c > 0`, то `a*c > b*c`.
Приклад: `7 > 4`, тому `7*2 > 4*2`, тобто `14 > 8`.

**Властивість 3 — множення на від'ємне число:**
**Знак нерівності змінюється на протилежний!**
Якщо `a > b` і `c < 0`, то `a*c < b*c`.
Приклад: `7 > 4`, але `7*(-2) < 4*(-2)`, тобто `-14 < -8`.

## Розв'язання лінійних нерівностей

Алгоритм той самий що для рівнянь — але при множенні або діленні на від'ємне число знак змінюється.

**Приклад: `3x + 5 > 14`**
- Віднімаємо 5: `3x > 9`
- Ділимо на 3: `x > 3`

**Приклад: `-2x + 1 >= 7`**
- Віднімаємо 1: `-2x >= 6`
- Ділимо на -2 (знак змінюється!): `x <= -3`

**Приклад: `4 - 3x < 13`**
- Віднімаємо 4: `-3x < 9`
- Ділимо на -3 (знак змінюється!): `x > -3`

## Подвійні нерівності

`-2 < x + 1 <= 5`
- Віднімаємо 1 з усіх частин: `-3 < x <= 4`
- Відповідь: x належить (-3; 4]

## Практичне застосування

Температура комфорту: `18 <= T <= 26`

Бюджет не більше 5000 грн: `x <= 5000`

Безпечна напруга для людини: `U < 36`

## Як перевірити чи правильно порахував?

Після того як знайшов суму — завжди корисно **перевірити** себе. Є кілька способів.

## Спосіб 1 — Переставити доданки

Якщо порахував **23 + 45 = 68**, перевір навпаки:

**45 + 23 = ?**

Якщо відповідь та сама — все правильно!

45 + 23 = 40 + 20 + 5 + 3 = 60 + 8 = **68** ✓

## Спосіб 2 — Порахувати ще раз іншим методом

Порахував у рядок — перевір у стовпчик. Або навпаки.

**Приклад: 37 + 46**

По частинах: 30 + 40 = 70, 7 + 6 = 13, разом 70 + 13 = **83**

У стовпчик:
```
  37
+ 46
----
  83
```
7 + 6 = 13 → пишемо 3, переносимо 1
3 + 4 + 1 = 8
Відповідь: **83** ✓

## Спосіб 3 — Округлення для перевірки

Округли числа і приблизно оціни відповідь.

**Приклад: 48 + 33**
- 48 ≈ 50, 33 ≈ 30
- Приблизно: 50 + 30 = 80
- Точна відповідь: 48 + 33 = **81**
- 81 близько до 80 — схоже на правду ✓

Якщо точна відповідь дуже далека від приблизної — десь помилка!

## Типові помилки

**Помилка 1 — забули перенести десяток:**
- Неправильно: 37 + 46 = 73 ✗
- Правильно: 37 + 46 = 83 ✓ (7 + 6 = 13, перенос!)

**Помилка 2 — переплутали одиниці і десятки:**
- Неправильно: 23 + 45 = 86 ✗
- Правильно: 23 + 45 = 68 ✓

**Помилка 3 — арифметична помилка:**
- Неправильно: 8 + 7 = 16 ✗
- Правильно: 8 + 7 = 15 ✓

## Золоте правило

Завжди перевіряй відповідь хоча б одним способом. Навіть найкращі математики роблять помилки!

## Координатна пряма

Числову пряму з від'ємними числами називають **координатною прямою**.

Кожна точка на ній має **координату** — число яке показує її положення.

- Точка A має координату 3 → A(3)
- Точка B має координату −5 → B(−5)
- Відстань між A і B: |3 − (−5)| = |8| = **8**

**Формула відстані між двома точками на прямій:**
d = |x₂ − x₁|

## Координатна площина

Якщо додати другу координатну пряму **перпендикулярно** до першої — отримаємо **координатну площину**.

- Горизонтальна пряма — **вісь X** (або абсциса)
- Вертикальна пряма — **вісь Y** (або ордината)
- Точка перетину — **початок координат O(0, 0)**

Кожна точка на площині задається **парою чисел (x, y)**:
- x — відстань по горизонталі від O
- y — відстань по вертикалі від O

## Чотири чверті (квадранти)

Осі ділять площину на **4 квадранти**:

**I квадрант:** x > 0, y > 0 (праворуч і вгору)
**II квадрант:** x < 0, y > 0 (ліворуч і вгору)
**III квадрант:** x < 0, y < 0 (ліворуч і вниз)
**IV квадрант:** x > 0, y < 0 (праворуч і вниз)

## Як нанести точку на площину?

Точка **A(3, −2)**:
1. Починаємо з початку координат O(0, 0)
2. Рухаємось по осі X на **+3** (вправо)
3. Рухаємось по осі Y на **−2** (вниз)
4. Ставимо точку

Точка **B(−4, 5)**:
1. По осі X на **−4** (вліво)
2. По осі Y на **+5** (вгору)
3. Ставимо точку

## Відстань між двома точками на площині

Для точок A(x₁, y₁) і B(x₂, y₂):

**d = √((x₂ − x₁)² + (y₂ − y₁)²)**

Це теорема Піфагора застосована до координат!

**Приклад:** відстань між A(1, 2) і B(4, 6):
- d = √((4−1)² + (6−2)²)
- d = √(9 + 16)
- d = √25 = **5**

## Де використовується координатна площина?

📍 **Карти і навігація**
GPS координати — це пара чисел (широта, довгота). Київ: (50.45, 30.52).

🎮 **Ігри і графіка**
Кожен піксель на екрані має координати (x, y). Позиція персонажа — теж координати.

📊 **Графіки функцій**
Будь-який графік — це набір точок на координатній площині.

🤖 **Робототехніка**
Положення руки робота задається координатами в просторі.

## Симетрія відносно осей

**Симетрія відносно осі X:** змінюємо знак y
- A(3, 4) → A'(3, −4)

**Симетрія відносно осі Y:** змінюємо знак x
- A(3, 4) → A'(−3, 4)

**Симетрія відносно початку координат:** змінюємо обидва знаки
- A(3, 4) → A'(−3, −4)

## Арифметика з від'ємними числами

Додавання і віднімання від'ємних чисел спочатку здається незвичним. Але є чіткі правила які завжди працюють.

## Додавання від'ємного числа

**Додати від'ємне = відняти додатне**

5 + (−3) = 5 − 3 = **2**

−4 + (−6) = −4 − 6 = **−10**

Інтуїція: якщо у тебе 500 грн і ти "додаєш" борг −200 грн — у тебе залишається 300 грн.

## Віднімання від'ємного числа

**Відняти від'ємне = додати додатне**

5 − (−3) = 5 + 3 = **8**

−4 − (−6) = −4 + 6 = **2**

Інтуїція: "відняти борг" означає позбутися боргу — ти стаєш багатшим.

## Правила знаків

При спрощенні двох знаків поруч:

- **(+)(+) = +** : 5 + (+3) = 5 + 3 = **8**
- **(+)(−) = −** : 5 + (−3) = 5 − 3 = **2**
- **(−)(+) = −** : 5 − (+3) = 5 − 3 = **2**
- **(−)(−) = +** : 5 − (−3) = 5 + 3 = **8**

Просте правило: **однакові знаки дають плюс, різні — мінус**.

## Додавання на числовій прямій

Числова пряма — найкращий інструмент для розуміння:

- Додавання додатного → рух **вправо**
- Додавання від'ємного → рух **вліво**

**3 + (−5):**
- Починаємо на 3
- Рухаємось вліво на 5
- Опиняємось на −2

**−2 + 7:**
- Починаємо на −2
- Рухаємось вправо на 7
- Опиняємось на 5

## Приклади
```
  8 + (−3)  = 8 − 3   = 5
 −5 + (−4)  = −5 − 4  = −9
  2 − (−7)  = 2 + 7   = 9
 −6 − (−2)  = −6 + 2  = −4
 −3 + 10    = 7
  4 − 9     = −5
```

## Реальні приклади

🌡️ **Температура**
Вночі було −12°C, вдень потеплішало на 18°C. Яка денна температура?
−12 + 18 = **6°C**

💰 **Фінанси**
Рахунок −3500 грн (борг). Надійшло 5000 грн. Який баланс?
−3500 + 5000 = **1500 грн**

📉 **Зміна висоти**
Підводний човен на глибині −200 м піднявся на 80 м. Де він тепер?
−200 + 80 = **−120 м**

## Властивості

Всі властивості додавання зберігаються:

**Переставна:** a + b = b + a
- (−3) + 7 = 7 + (−3) = 4

**Сполучна:** (a + b) + c = a + (b + c)
- (−2 + 5) + (−3) = −2 + (5 + (−3)) = 0

**Нейтральний елемент:** a + 0 = a
- (−7) + 0 = −7

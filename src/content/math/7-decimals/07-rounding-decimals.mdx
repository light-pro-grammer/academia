## Навіщо округляти?

Іноді точне число не потрібне — достатньо **приблизного**:
- Ціна 47,99 гривень ≈ **48** гривень
- Відстань 3,47 км ≈ **3,5** км
- Вага 2,134 кг ≈ **2,1** кг

## Правило округлення

Округлення до **цілого числа** — дивимось на **десяті** (перша цифра після коми):
- Десяті **менше 5** → округляємо **вниз** (відкидаємо дробову частину)
- Десяті **5 або більше** → округляємо **вгору** (додаємо 1 до цілої частини)

**Приклади:**
- 3,**2** → **3** (2 < 5, вниз)
- 7,**8** → **8** (8 ≥ 5, вгору)
- 5,**5** → **6** (5 = 5, вгору)
- 12,**4** → **12** (4 < 5, вниз)

## Округлення до десятих

Дивимось на **соті** (друга цифра після коми):
- Соті **менше 5** → відкидаємо соті
- Соті **5 або більше** → збільшуємо десяті на 1

**Приклади:**
- 3,4**2** → **3,4** (2 < 5, вниз)
- 7,8**7** → **7,9** (7 ≥ 5, вгору)
- 5,5**5** → **5,6** (5 = 5, вгору)
- 2,3**4** → **2,3** (4 < 5, вниз)

## Округлення до сотих

Дивимось на **тисячні** (третя цифра після коми):

**Приклади:**
- 3,14**1** → **3,14** (1 < 5, вниз)
- 2,71**8** → **2,72** (8 ≥ 5, вгору)
- 1,99**5** → **2,00** (5 = 5, вгору — і переносимо!)

📏 **Вимірювання:** 1,756 м ≈ 1,76 м (до сотих — сантиметрів)

🔢 **Число π:** 3,14159... ≈ 3,14 (до сотих)

## Важливо

Округлення — це **наближення**, не точна відповідь. Завжди вказуй до якого розряду округлив:
- "Округлено до цілого: 3"
- "Округлено до десятих: 3,5"

## Додаємо десяткові дроби

Додавання десяткових дробів схоже на додавання цілих чисел — але є один важливий момент!

## Головне правило — коми під комами!

Записуй приклади у стовпчик так щоб **коми стояли одна під одною**. Тоді автоматично десяті під десятими, соті під сотими.

**Приклад: 3,4 + 2,5**
```
  3,4
+ 2,5
-----
  5,9
```

Рахуємо справа наліво:
- Десяті: 4 + 5 = **9**
- Цілі: 3 + 2 = **5**
- Відповідь: **5,9**

## Приклад з переносом

**Приклад: 2,7 + 1,8**
```
  2,7
+ 1,8
-----
  4,5
```

- Десяті: 7 + 8 = 15 → пишемо 5, переносимо 1
- Цілі: 2 + 1 + 1 (перенос) = **4**
- Відповідь: **4,5**

## Різна кількість знаків після коми

Якщо у чисел **різна** кількість знаків після коми — вирівнюємо нулями:

**Приклад: 3,5 + 1,47**
```
  3,50
+ 1,47
------
  4,97
```

- 3,5 → 3,50 (додаємо нуль)
- Соті: 0 + 7 = **7**
- Десяті: 5 + 4 = **9**
- Цілі: 3 + 1 = **4**
- Відповідь: **4,97**

## Додавання з цілим числом

**Приклад: 5 + 2,3**
```
  5,0
+ 2,3
-----
  7,3
```

- 5 = 5,0 (додаємо кому і нуль)
- Відповідь: **7,3**

## Перевірка

Завжди перевіряй приблизним підрахунком:
- 3,4 + 2,5 ≈ 3 + 3 = 6 (точна відповідь 5,9 — схоже ✓)
- 2,7 + 1,8 ≈ 3 + 2 = 5 (точна відповідь 4,5 — схоже ✓)

```
  45,50
+ 32,75
-------
  78,25
```
Відповідь: **78,25** гривень

📏 Довжина першого шматка дроту 2,4 м, другого — 1,85 м. Яка загальна довжина?
```
  2,40
+ 1,85
------
  4,25
```
Відповідь: **4,25** метра

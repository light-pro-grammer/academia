## Що таке квадратне рівняння?

**Квадратне рівняння** — рівняння виду:

**ax² + bx + c = 0**

де a, b, c — коефіцієнти, a != 0.

Приклади:
- x² - 5x + 6 = 0 (a=1, b=-5, c=6)
- 2x² + 3x - 2 = 0 (a=2, b=3, c=-2)
- x² - 9 = 0 (a=1, b=0, c=-9)
- x² + 4x = 0 (a=1, b=4, c=0)

## Дискримінант

**Дискримінант** — ключова величина яка визначає кількість коренів:

**D = b² - 4ac**

- D `>` 0 → **два різних** дійсних кореня
- D = 0 → **один** дійсний корінь (кратний)
- D `<` 0 → **немає** дійсних коренів (є комплексні)

## Формула коренів

При D `>` 0 або D = 0:

**x = (-b ± sqrt(D)) / 2a**

Тобто:
- x₁ = (-b + sqrt(D)) / 2a
- x₂ = (-b - sqrt(D)) / 2a

При D = 0: x₁ = x₂ = -b / 2a

## Приклад 1 — два корені

**x² - 5x + 6 = 0**

a=1, b=-5, c=6

D = (-5)² - 4*1*6 = 25 - 24 = **1**

D `>` 0 → два кореня:
- x₁ = (5 + 1) / 2 = **3**
- x₂ = (5 - 1) / 2 = **2**

Перевірка: (x-3)(x-2) = x² - 5x + 6 ✓

## Приклад 2 — один корінь

**x² - 6x + 9 = 0**

D = 36 - 36 = **0**

x = 6/2 = **3**

Перевірка: (x-3)² = x² - 6x + 9 ✓

## Приклад 3 — немає коренів

**x² + x + 1 = 0**

D = 1 - 4 = **-3**

D `<` 0 → дійсних коренів немає.

## Теорема Вієта

Для рівняння x² + px + q = 0 з коренями x₁ і x₂:

- **x₁ + x₂ = -p = -b/a** (сума коренів)
- **x₁ * x₂ = q = c/a** (добуток коренів)

**Приклад:** x² - 5x + 6 = 0
- Сума: x₁ + x₂ = 5 (і дійсно: 3 + 2 = 5 ✓)
- Добуток: x₁ * x₂ = 6 (і дійсно: 3 * 2 = 6 ✓)

Теорема Вієта дозволяє швидко перевірити відповідь або підібрати корені.

## Розкладання на множники

Якщо знайдено корені x₁ і x₂:

ax² + bx + c = a(x - x₁)(x - x₂)

**Приклад:** x² - 5x + 6 = (x - 3)(x - 2)

## Неповні квадратні рівняння

**b = 0:** ax² + c = 0
- x² = -c/a
- Корені: x = ±sqrt(-c/a) якщо -c/a `>` 0

Приклад: x² - 16 = 0 → x = ±4

**c = 0:** ax² + bx = 0
- x(ax + b) = 0
- Корені: x = 0 або x = -b/a

Приклад: x² + 3x = 0 → x(x+3) = 0 → x = 0 або x = -3
